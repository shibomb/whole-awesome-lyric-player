Put font files in this directory.
and set `ALP_CONFIG.STYLE.FONT_FILENAME` to the font file name.

# Recommended Fonts

## BIZ UD Gothic
Description: Universal Design Gothic
Filename: BIZUDPGothic-Regular.ttf
Download: [Google Fonts: BIZ UD Gothic](https://fonts.google.com/specimen/BIZ+UDGothic)

## Murecho
Description: Universal Design Gothic (Balanced with Alphanumerics, but font outline is not good.)
Note: TEXT_STROKE_WEIGHT should be 0.
Filename: Murecho-Regular.ttf
Download: [Google Fonts: Murecho](https://fonts.google.com/specimen/Murecho)

## しねきゃぷしょん
Description: Cinema Caption
Filename: cinecaption227.ttf
Download: [Vector しねきゃぷしょん](https://www.vector.co.jp/soft/data/writing/se314690.html)

## Kosugi Maru
Description: Maru Gothic
Filename: KosugiMaru-Regular.ttf
Download: [Google Fonts: Kosugi Maru](https://fonts.google.com/specimen/Kosugi+Maru)
